## SECTION 8: API ENDPOINTS REFERENCE
====================================================================================================

### Core Fundamentals Endpoint:
```
https://eodhd.com/api/fundamentals/{TICKER}.{EXCHANGE}?api_token={KEY}&fmt=json
```

### Filtered Requests (reduce payload):
```
# Single section
https://eodhd.com/api/fundamentals/AAPL.US?filter=General&api_token={KEY}&fmt=json

# Nested field
https://eodhd.com/api/fundamentals/AAPL.US?filter=General::Code&api_token={KEY}&fmt=json

# Multiple sections
https://eodhd.com/api/fundamentals/AAPL.US?filter=General,Highlights,Valuation&api_token={KEY}&fmt=json
```

### Index Constituents (with historical):
```
# Current constituents only
https://eodhd.com/api/fundamentals/GSPC.INDX?api_token={KEY}&fmt=json

# With historical snapshots (from 1960s for S&P 500)
https://eodhd.com/api/fundamentals/GSPC.INDX?historical=1&from=2020-01-01&to=2023-01-01&api_token={KEY}&fmt=json
```

### ETF & Mutual Fund Fundamentals:
```
# ETF fundamentals
https://eodhd.com/api/fundamentals/SPY.US?api_token={KEY}&fmt=json

# Mutual fund fundamentals
https://eodhd.com/api/fundamentals/SWPPX.US?api_token={KEY}&fmt=json
```
