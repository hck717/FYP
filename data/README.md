# Local Data Storage
This directory is for local data persistence.

## Contents
- **/raw**: Raw downloaded files (git-ignored).
- **/logs**: System logs (git-ignored).
- **/postgres_data**: Mounted volume for PostgreSQL (git-ignored).
- **/qdrant_storage**: Mounted volume for Qdrant (git-ignored).

**Note:** Do not commit sensitive data or large datasets to the repository.
